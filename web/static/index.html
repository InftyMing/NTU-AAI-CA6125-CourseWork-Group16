Course project · Feedback Prize · English Language Learning

Six writing scores for every English learner essay, in one reproducible pipeline.

We train transparent linear and tree-based models on the ELLIPSE corpus, blend their out-of-fold predictions per target, and expose the result through this demo. Every number on this page comes from a real cross-validation run, not a hand-picked submission.

Pipeline at a glance

data → features → component models → stacked ensemble → submission
01

Audit & split

Verify schema, compute text statistics, build deterministic stratified folds on the rounded mean target.

02

Feature views

Word + character TF-IDF, ten interpretable text statistics, and a 128-dim Truncated SVD projection.

03

Component models

Per-target Ridge, Ridge with fused features, and LightGBM on SVD + statistics.

04

Stacked blend

Convex combination per target, fitted on out-of-fold predictions to avoid leakage.

05

Submission

Clipped scores written to submission.csv and reproduced inside Group16_inference.ipynb.
